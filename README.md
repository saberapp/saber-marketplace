# Arsenal Marketplace

This is the official marketplace catalog for **Arsenal** — GTM and RevOps skills for Claude Code, built by [Saber](https://saber.app).

Arsenal brings revenue intelligence directly into your Claude Code workflow: prospecting, signal tracking, account research, and outreach — all without leaving your terminal.

Source repository: [https://github.com/saberapp/arsenal](https://github.com/saberapp/arsenal)

---

## Installation

### Step 1 — Add this marketplace (one-time setup)

```
/plugins add-marketplace saberapp/arsenal-marketplace
```

### Step 2 — Install Arsenal

```
/plugins install arsenal
```

That's it. Arsenal skills will be available in your next Claude Code session.

---

## What's included

Arsenal ships a set of skills that integrate with the Saber platform:

- **Signal discovery** — define buying intent signals that match your ICP
- **Account list building** — build and qualify target account lists
- **Contact list building** — build and qualify target contact lists
- **Company signals** — activate company-level signal tracking
- **Contact signals** — activate contact-level signal tracking
